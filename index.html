CasparCG config file validator

This is a simple validator for CasparCG config files.

Drag your casparcg.config file into the box below or paste in the contents of your config file below and hit . The results will display below.

Validates XML, verifies that only known tags are used, checks if user is using example code, verifies values of some tags. Summary with the number of channels and configured outputs.
